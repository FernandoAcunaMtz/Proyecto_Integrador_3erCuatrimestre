Bienvenido a HTML y CSS Avanzado
Domina las técnicas avanzadas de maquetación y diseño web moderno. Aprende a crear interfaces hermosas y responsivas con las últimas tecnologías CSS.
¿Qué aprenderás?
  • Flexbox Avanzado y CSS Grid Layout
  • Media Queries y Diseño Responsive
  • Animaciones y Transiciones CSS
  • Metodologías CSS (BEM, SMACSS)
  • Preprocesadores (Sass/SCSS)
  • CSS-in-JS y Frameworks Modernos
  • Optimización y Performance
Requisitos
  • Conocimientos básicos de HTML y CSS
  • Un editor de código (VS Code recomendado)
  • Navegador web moderno
  • Conocimientos básicos de JavaScript (recomendado)
