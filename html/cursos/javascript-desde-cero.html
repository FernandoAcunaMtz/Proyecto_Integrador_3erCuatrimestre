Bienvenido a JavaScript desde Cero
En este curso aprenderás los fundamentos de JavaScript desde lo más básico hasta conceptos avanzados. JavaScript es el lenguaje más popular del mundo web y es esencial para cualquier desarrollador frontend.
¿Qué aprenderás?
  • Variables y tipos de datos
  • Funciones y scope
  • Manipulación del DOM
  • Eventos y listeners
  • Async/Await y Promises
  • ES6+ Features
  • Módulos y bundlers
Requisitos
  • Conocimientos básicos de HTML
  • Un editor de código (VS Code recomendado)
  • Navegador web moderno
